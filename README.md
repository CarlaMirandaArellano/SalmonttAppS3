   En el proyecto SalmonttApp se muestra la aplicación práctica de:
* Interfaces y contratos de comportamiento
* Herencia y polimorfismo
* Uso de colecciones dinámicas
* Validación de tipos en tiempo de ejecución
* Integración de una interfaz gráfica funcional
Todo esto orientado a simular un sistema real, usable y escalable, adecuado para el contexto de una empresa salmonera.
